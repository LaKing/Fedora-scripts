Fedora-scripts
==============

Bash scripts for Fedora based systems.

Wokstation installer can be used to add additional software automatically. Browsers, media players, etc.
Status: production-ready. I installed already several systems. 
If you notice something, or you have an improvement, please let me know.

The playlist-player is a script that will allow you to play all songs randomly, without repetitions in three lanes.

As of 2.x srvctl moved to it's own repository.
https://github.com/LaKing/srvctl
